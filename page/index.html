Abstract

Imitation learning provides a promising paradigm for acquiring complex manipulation skills from expert demonstrations. However, deploying learned policies for deformable object manipulation remains challenging due to the objects' infinite-dimensional state spaces and the difficulty of recovering from execution errors.

We present PHASER (Phase-Aware Self-Recovering Imitation Learning), a hierarchical framework that addresses these challenges through three key innovations: (1) a phase-based task decomposition that explicitly structures the manipulation process into semantically meaningful stages, (2) a force-aware policy conditioned on tactile feedback for robust interaction with deformable objects, and (3) a closed-loop recovery mechanism that detects failures via phase prediction and autonomously re-routes execution to the appropriate recovery behavior.

Method

PHASER decomposes the manipulation task into a sequence of explicit phases, each associated with a force-aware sub-policy. A lightweight phase predictor continuously monitors the task progress. When the predicted phase deviates from the expected execution flow, the system triggers a recovery policy — closing the loop between perception, prediction, and action.

Key Contributions:

  • Phase Predictor: A lightweight network that maps observations to discrete phase labels, providing interpretable task progress monitoring.
  • Force-Aware Policy: Each phase-specific policy is conditioned on force/torque readings, enabling compliant manipulation of deformable objects.
  • Recovery Mechanism: Phase transition anomalies automatically trigger recovery sub-policies that restore valid manipulation states.

Results

We evaluate PHASER on a diverse set of deformable object manipulation tasks, comparing against state-of-the-art imitation learning baselines. We assess both nominal task success and recovery ability under injected perturbations.
